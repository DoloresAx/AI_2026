### Feature Branch Workflow ကို အသုံးပြုပါ။
```
# first fork repo ( or use gh cli on machine )
# and clone your repo to local machine
# checkout new_branch
# do your work
# commit and push to your repo
# send pr to merge ( or use gh cli from local machine )
```

### သင်တန်းသားများသတိပြုရန်
```
- code မရေးမီ အမြဲတမ်း sync and git pull လုပ်ပါ။
- ပြီးနောက် branch အသစ် ခွဲပြီးရေးပါ။
- အဆင်ပြေတာနဲ့ commit, merge, push ပါ။
- remote နှစ်ခုရှိရင် ဘယ်ကို push လည်း သတိပြုပါ။ ( upstream or origin ) 
- မိမိရဲ့ repo ကို သာ push လုပ်ပြီးနောက် မလိုတော့တဲ့ branch ကို delete လုပ်ပါ။
- upstream ဆီသို pr တင်လို့ရပါပြီ။
- အခြား dev team များလည်းရှိနိုင်တာမို့ pr , merge ပြီးပါက sync ပြန်လုပ်ထားပါ။
```
